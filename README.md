# Project UAS Data Warehouse Olist

Project ini berisi perancangan dan implementasi data warehouse untuk pengelolaan data penjualan Brazilian E-Commerce by Olist menggunakan Star Schema dan proses ETL dengan Pentaho Data Integration.

## Tools
- Pentaho Data Integration / Spoon
- MySQL
- phpMyAdmin
- GitHub

## Isi Repository
- Folder dataset berisi link data dari Kaggle.
- Folder etl berisi file transformation Pentaho (.ktr) dan job (.kjb).
- Folder sql berisi query pembuatan database oltp, database data warehouse dan query KPI.
- Folder laporan berisi dokumentasi project dalam format PDF.

## Dataset
Dataset yang digunakan adalah Brazilian E-Commerce Public Dataset by Olist dari Kaggle.
Link: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
